## 네이밍 규칙

- **HTML**
  - id: camelCase
  - name: camelCase
  - class: `camelElement_camelModifier`
- **CSS**
  - 변수: kebab-case
- **JS**
  - 식별자: camelCase
  - 타입: PascalCase

## 프로젝트 구조

```
codeit-fe10-sprint-mission
├─ .github                   : GitHub 설정
├─ public                    : 정적 파일
│   ├─ fonts                 : 글꼴
│   ├─ icons                 : 아이콘
│   ├─ images                : 이미지
│   └─ meta                  : 메타데이터
└─ src                       : 소스 코드
    ├─ apis                  : 통신 API
    ├─ components            : 컴포넌트
    │   ├─ common            : 공용 컴포넌트
    │   └─ layout            : 레이아웃 컴포넌트
    ├─ pages                 : 페이지
    └─ styles                : 전역 CSS
```
