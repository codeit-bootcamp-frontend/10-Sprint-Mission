## 프로젝트 구조

```
step2-FE-kakao-shop         : 프로젝트 디렉터리
├─ .github                  : GitHub 설정
├─ public                   : Non-Compile 파일
├─ readme                   : README.md 미디어
└─ src                      : Compile 파일
    ├─ apis                 : 통신 API
    ├─ assets               : 미디어
    ├─ components           : 컴포넌트
    ├─ constants            : 상수
    ├─ contexts             : 전역 상태
    ├─ hooks                : 커스텀 훅
    ├─ pages                : 페이지
    ├─ styles               : CSS
    └─ utils                : 유틸리티 함수
```
